
========================================================
LOGOS — DATABASE STRUCTURE
File: LOGOS_ENGINE_DATABASE_v1.0
Tipo: Google Sheets / Excel
Architettura: Event Ledger + Dataset Analitici
========================================================

TABELLA: RAW_DATA
Scopo: Ledger eventi del sistema.

Campi:
ID_Movimento (PK)
Timestamp
Data_Evento
Soggetto_Input
Project_ID (FK → PROJECTS.Project_ID)
Entity_ID (FK → ENTITIES.Entity_ID)
Tipo_Movimento (Tempo / Spesa / Incasso / Evento)
Valore
Unita
Causale
Categoria_Derivata
Riferimento_ID
Fonte
Metodo_Pagamento
Note

--------------------------------------------------------

TABELLA: PROJECTS
Scopo: Anagrafica progetti e gerarchie.

Campi:
Project_ID (PK)
Project_Name
Parent_Project_ID (FK → PROJECTS.Project_ID)
Project_Type
Status
Created_Timestamp
Note

Relazioni:
Project_ID → RAW_DATA.Project_ID
Parent_Project_ID → PROJECTS.Project_ID

--------------------------------------------------------

TABELLA: ENTITIES
Scopo: Anagrafica soggetti operativi.

Campi:
Entity_ID (PK)
Entity_Type
Display_Name
Entity_Status
Email
Telefono
Parent_Entity_ID (FK → ENTITIES.Entity_ID)
Note
Created_Timestamp
Modified_Timestamp

Relazioni:
Entity_ID → RAW_DATA.Entity_ID
Parent_Entity_ID → ENTITIES.Entity_ID

--------------------------------------------------------

TABELLA: SETTINGS
Scopo: configurazioni sistema e liste valori.

Campi:
Setting_Type
Value
Description

Liste:
Project_Type_List
Status_List
Entity_Type_List
Entity_Status_List
Entity_ID_List

--------------------------------------------------------

TABELLA: ENTITY_CONFIRMATION
Scopo: risoluzione entità per evitare duplicati.

Campi:
Input_Text
Suggested_Entity
Suggested_Entity_ID
User_Confirmation
Confirmed_Entity_ID
Timestamp
Status

--------------------------------------------------------

TABELLA: SYSTEM_LOG
Scopo: log tecnico del sistema.

Campi:
Timestamp
Event_Type
Sheet
Row
User
Details

Eventi:
NEW_MOVEMENT
PROJECT_CREATED
ENTITY_CREATED
PRIMARY_KEY_PROTECTION
PROJECT_HIERARCHY_ERROR
ENTITY_HIERARCHY_ERROR
ENTITY_CONFIRMATION_REQUIRED

--------------------------------------------------------

DATASET ANALITICI

CONTROLLO_PROJECT
CONTROLLO_PROJECT_DETTAGLIO
CONTROLLO_ENTITY
CONTROLLO_ENTITY_DETTAGLIO
